# Bank Account Management System

A comprehensive Python banking system built with Jupyter Notebook.

## Features
- Create and manage bank accounts (Savings/Current)
- Perform transactions (Deposit, Withdraw, Transfer)
- Transaction history tracking
- Statistical analysis using NumPy
- Data persistence with pickle
- Interactive menu system

## Technologies Used
- Python 3
- NumPy
- Pickle (file handling)
- Jupyter Notebook

## How to Run
1. Clone the repository
2. Open `Bank_Account_Management_System.ipynb` in Jupyter
3. Run cells 1-3 to load classes
4. Run cell 4 for demo or cell 5 for interactive menu
